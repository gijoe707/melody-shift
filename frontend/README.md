# Melody Shift - Frontend

The modern, responsive web interface for Melody Shift, built for speed and user experience.

## 🛠️ Tech Stack

- **Build Tool:** Vite
- **Framework:** React (TypeScript)
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI, Radix UI
- **Icons:** Lucide React
- **State Management:** React Query (@tanstack/react-query)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Development

Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

## ✨ Key Features

- **Modern UI:** Clean, glassmorphic design with smooth animations.
- **Real-time Progress:** Visual feedback for playlist transfers.
- **Responsive:** Works seamlessly on desktop and mobile.
- **Interactive:** Drag-and-drop, easy selection, and instant feedback.
